# Java
- objektová
- interpretovaná
- `.java` -> *compilation* -> `.class` **bytecode** -> *interpretace virtuálním strojem*
- JIT - kompilace bytecode do stojoveho kodu behem behu
- multiplatformní
- ale programy žerou hrozně moc paměti oproti třeba C++
- stařenka
- skoro backward compatible
